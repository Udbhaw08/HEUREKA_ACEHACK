## External Services

### GitHub API
- **Purpose:** Repository analysis, commit history, language detection
- **Authentication:** Personal Access Token (PAT)
- **Rate Limit:** 5000 requests/hour (authenticated)
- **Endpoint:** https://api.github.com
- **Documentation:** https://docs.github.com/en/rest

### LeetCode
- **Purpose:** Problem-solving verification, contest ratings
- **Authentication:** Session cookies (web scraping)
- **Rate Limit:** Aggressive throttling, no official API
- **Endpoint:** https://leetcode.com
- **Method:** Web scraping with BeautifulSoup/Selenium

### Codeforces API
- **Purpose:** Competitive programming ratings, submissions
- **Authentication:** None (public API)
- **Rate Limit:** 1 request per 2 seconds
- **Endpoint:** https://codeforces.com/api
- **Documentation:** https://codeforces.com/apiHelp

### LinkedIn
- **Purpose:** Professional history extraction
- **Authentication:** None (PDF upload)
- **Method:** PDF parsing with PyPDF2/PyMuPDF
- **Note:** No API access, manual PDF upload required

### OpenAI
- **Purpose:** LLM for text analysis, bias detection, matching
- **Authentication:** API key
- **Model:** GPT-4 or GPT-3.5-turbo
- **Endpoint:** https://api.openai.com/v1

### Ollama (Local)
- **Purpose:** Local LLM alternative to OpenAI
- **Authentication:** None (local)
- **Model:** claude-3-haiku (or other local models)
- **Endpoint:** http://localhost:11434
