# Frontend – Fair Hiring Platform

React frontend providing candidate & company dashboards.

---

## 🚀 Run Frontend

```bash
npm install
npm run dev
```

Runs at:

```
http://localhost:5173
```

---

## 👤 Candidate Views

- Signup / Login
- Available roles
- Apply to jobs
- Application status
- Skill passport
- Feedback & scores

---

## 🏢 Company Views

- Signup / Login
- Job creation
- Dashboard metrics
- Role pipeline
- Run matching
- Selected candidates

---

## 🔗 API Integration

Frontend consumes real backend APIs:

- No mock data
- No hardcoded counts
- Live stats from database

---

## 🎨 Design System

- Consistent box-based UI
- Shared components
- No theme toggles (single theme)
